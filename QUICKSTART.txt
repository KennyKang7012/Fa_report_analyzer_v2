📦 專案簡介
────────────────────────────────────────────────────────────────
FA Report Analyzer 是一個專業的失效分析報告評估工具,使用 Claude AI 
技術對 FA 報告進行全面、客觀的評分與分析。

🎯 核心功能
────────────────────────────────────────────────────────────────
✅ 自動讀取多種格式 (TXT, PDF, DOCX)
✅ AI 智能分析與評分
✅ 六大維度全面評估
✅ 自動生成評估報告
✅ 具體改善建議

📋 評估維度 (總分 100)
────────────────────────────────────────────────────────────────
1. 基本資訊完整性 (15%) - 產品資訊、客戶資訊、FA 編號等
2. 問題描述與定義 (15%) - 失效現象、失效模式、影響評估
3. 分析方法與流程 (20%) - 分析方法適當性、步驟完整性
4. 數據與證據支持 (20%) - 數據充分性、圖表清晰度
5. 根因分析 (20%) - 根本原因深度、邏輯推導
6. 改善對策 (10%) - 短中長期對策、可行性

⚡ 快速開始 (3 步驟)
────────────────────────────────────────────────────────────────

步驟 1: 安裝依賴
──────────────
pip install anthropic pandas --break-system-packages

步驟 2: 準備報告
──────────────
將你的 FA 報告放在同一目錄下 (支援 .txt, .pdf, .docx)

步驟 3: 執行分析
──────────────
python fa_report_analyzer.py -i your_fa_report.txt

🎓 使用範例
────────────────────────────────────────────────────────────────

範例 1: 基本使用
────────────────
python fa_report_analyzer.py -i sample_fa_report.txt

範例 2: 指定輸出文件
───────────────────
python fa_report_analyzer.py -i report.pdf -o evaluation.txt

範例 3: 使用 API key (本地環境)
──────────────────────────────
python fa_report_analyzer.py -i report.pdf -k YOUR_API_KEY

範例 4: 測試範例報告
────────────────────
python usage_examples.py

📊 評分等級
────────────────────────────────────────────────────────────────
A 級 (90-100 分) → 卓越報告
B 級 (80-89 分)  → 良好報告  
C 級 (70-79 分)  → 合格報告
D 級 (60-69 分)  → 待改進報告
F 級 (<60 分)    → 不合格報告

💡 實用技巧
────────────────────────────────────────────────────────────────

技巧 1: 批次分析多份報告
────────────────────────
for file in *.txt; do
    python fa_report_analyzer.py -i "$file" -o "eval_$file"
done

技巧 2: 查看幫助訊息
────────────────────
python fa_report_analyzer.py --help

技巧 3: 使用程式化接口
─────────────────────
from fa_report_analyzer import FAReportAnalyzer
analyzer = FAReportAnalyzer()
result = analyzer.analyze_report('report.txt')
print(f"分數: {result['total_score']}, 等級: {result['grade']}")

🎯 輸出報告格式
────────────────────────────────────────────────────────────────

評估報告包含:
1. 總分與等級
2. 各維度評分表
3. 優點分析 (3-5 項)
4. 待改進項目 (依優先級)
5. 具體改善建議
6. 總評與建議

📊 輸出範例預覽
────────────────────────────────────────────────────────────────
【總分與等級】
總分: 85.5 分
等級: B 級 - 良好報告

【各維度評分表】
評估維度         權重  得分  完成度  評語
基本資訊完整性   15%   13.5  90.0%   資訊完整...
...

【優點分析】
1. 報告結構完整,包含所有必要章節
2. 根因分析使用了系統化的 5-Why 方法
3. 改善對策分為短中長期,具有可操作性

【待改進項目】
1. [高優先級] 缺少對照組數據
   改善建議: 建議增加良品的對照測試數據...

🛠️ 故障排除
────────────────────────────────────────────────────────────────

問題: 無法讀取 PDF
解決: pip install PyPDF2 --break-system-packages

問題: API key 錯誤
解決: 檢查 API key 是否正確,或在 claude.ai 環境中不需要

問題: 記憶體不足
解決: 將大型 PDF 轉換為文字格式

🚀 進階用法
────────────────────────────────────────────────────────────────

1. 自訂評估維度權重
   → 編輯 fa_report_analyzer.py 中的 dimensions 字典

2. 修改評分標準
   → 編輯 grade_criteria 字典

3. 整合到現有系統
   → 作為 Python 模組導入使用

4. 批次處理與統計
   → 使用 usage_examples.py 中的批次範例
