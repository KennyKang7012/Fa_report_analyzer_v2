本專案包含兩個版本的 FA 報告分析工具:
  • v1.0 - 基礎版本，僅支援 Anthropic Claude
  • v2.0 - 進階版本，支援 Ollama/OpenAI/Anthropic + 圖片分析

推薦: 優先使用 v2.0 + Ollama 享受免費、本地化和多模態分析!

🚀 推薦使用流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

第一步: 選擇版本
  推薦 → v2.0 + Ollama（免費、本地化、支援圖片）
  備選 → v2.0 + OpenAI（高精度，需付費）
  簡單 → v1.0（基礎功能，設置簡單）

第二步: 閱讀文檔
  必讀 → QUICKSTART_v2.txt（快速開始）
  選讀 → README_v2.md（完整功能）
  選讀 → OLLAMA_SETUP.md（Ollama 配置）

第三步: 安裝配置
  方案 A → ./install_ollama.sh（一鍵安裝）
  方案 B → 按照 QUICKSTART_v2.txt 手動安裝

第四步: 測試驗證
  基礎測試 → python fa_report_analyzer_v2.py -i sample_fa_report.txt
  功能測試 → python usage_examples.py

第五步: 實際使用
  單次分析 → python fa_report_analyzer_v2.py -i your_report.pdf
  批次處理 → 參考 usage_examples.py
